# Changelog

## 0.1.1 - 2026-07-23

- 한 계정으로 연결·시작·완료·주간 집계를 확인하는 상태 진단 추가
- `AI 원정대 연결 상태 확인해줘` 요청으로 서버 신호와 랭킹 반영 여부 확인
- 공식 `PLUGIN_DATA` 저장 위치를 우선 사용하고 기존 로컬 경로는 호환용으로 유지
- 원정 신호가 없을 때 Codex `/hooks` 신뢰 확인 안내

## 0.1.0 - 2026-07-23

- Codex 작업 시작·완료를 AI 원정대 서버에 기록
- 일회용 6자리 코드로 게임 계정 연결
- Windows PowerShell 및 macOS·Linux Node 실행 지원
- 질문·답변·파일·도구 출력을 전송하지 않는 최소 수집 방식
